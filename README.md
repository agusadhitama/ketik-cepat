## ✨ Fitur

### 🎮 Gameplay
- **Teks Bahasa Indonesia asli** kalimat sehari-hari, bukan terjemahan
- **3 tingkat kesulitan** Mudah 😌 · Sedang 😤 · Sulit 🔥
- **3 durasi tes** 30 detik · 60 detik · 2 menit
- **Auto-lanjut** teks baru otomatis muncul saat kamu selesai mengetik
- **Highlight real-time** karakter benar, salah, dan posisi kursor langsung terlihat

### 📊 Statistik Live
| Statistik | Keterangan |
|-----------|-----------|
| ⚡ KPM | Kata Per Menit, dihitung real-time |
| 🎯 Akurasi | Persentase karakter yang benar |
| ⏱ Waktu | Hitung mundur dengan indikator merah saat kritis |
| 🔥 Streak | Jumlah karakter benar berturut-turut |

### 🏆 Hasil & Riwayat
- **Sistem grading** S / A / B / C / D berdasarkan KPM dan akurasi
- **Modal hasil** dengan animasi pop setelah tes selesai
- **Riwayat 10 sesi** tersimpan otomatis di localStorage (tidak hilang saat refresh)
- **Ranking 🥇🥈🥉** otomatis diurutkan dari skor tertinggi

### 🎨 Desain
- Background krem hangat dengan dot grid pattern
- Palet warna: Coral · Teal · Gold · Violet
- Typography: **Playfair Display** + **JetBrains Mono** + **Plus Jakarta Sans**
- Micro-animations: card pulse, overlay float, modal pop
- Status badge dinamis berubah warna sesuai state

## 🚀 Cara Pakai

### Opsi 1 Download langsung
```bash
# Clone repo ini
git clone https://github.com/agusadhitama/ketik-cepat.git

# Buka file di browser
open index.html
```

### Opsi 2 Pakai langsung dari browser
Tidak perlu instalasi. Cukup buka `index.html` di browser manapun Chrome, Firefox, Safari, Edge.

## 📈 Sistem Grading

|-------|-----|---------|
| **S** 🌟 | ≥ 80 | ≥ 95% |
| **A** 🔥 | ≥ 60 | ≥ 90% |
